# Malabar Rasoi: Serving Heritage on Every Plate

## Project Overview
Malabar Rasoi is a vintage-themed restaurant website that combines traditional aesthetics with modern web technologies.

## Features
- Responsive website design
- Interactive digital menu
- Online table reservation form
- Restaurant gallery
- Customer testimonials
- Smooth navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
